# Paso a paso

## 0. Requisitos

- Node.js v18+
- NPM

## 1. Crear un nuevo proyecto

```bash
npx create-wagmi@latest
```
Selecciona las siguientes opciones:

```bash
✔ Project name: … web2-to-web3
✔ Select a framework: › React
✔ Select a variant: › Vite
```

## 2. Instalar dependencias

```bash
npm install
```
