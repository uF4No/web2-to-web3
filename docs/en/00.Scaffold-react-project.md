# Step-by-step

## 0. Requirements

- Node.js v18+
- NPM

## 1. Create a new project

```bash
npx create-wagmi@latest
```
Select the following options:

```bash
✔ Project name: … web2-to-web3
✔ Select a framework: › React
✔ Select a variant: › Vite
```

## 2. Install dependencies

```bash
npm install
```
